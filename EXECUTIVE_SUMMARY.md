## Overview

Drishti is a state-of-the-art AI-powered investigative platform designed specifically for the Karnataka State Police. The system leverages cutting-edge artificial intelligence, machine learning, and modern web technologies to transform crime investigation from a manual, time-consuming process into an intelligent, data-driven operation.

## What We Built

### Core Platform
A comprehensive web-based system that enables police officers to:
- **Query crime data in natural language** (English and Kannada)
- **Visualize criminal networks** and relationships
- **Predict crime trends** and identify hotspots
- **Generate professional reports** automatically
- **Access the system securely** with role-based permissions

### Technical Implementation
- **Frontend:** Modern React application with TypeScript
- **Backend:** High-performance FastAPI server with Python
- **AI/ML:** Mistral AI for intelligence, Prophet for predictions
- **Database:** PostgreSQL for structured data, ChromaDB for AI embeddings
- **Security:** JWT authentication, role-based access control, audit logging
